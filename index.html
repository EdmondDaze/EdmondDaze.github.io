Intro

Hello, thank you for visiting my page. My name is Elijah Orozco. I've been doing laser programming for about a year now and programming in code itself for about 8 years. I have mainly programmed in C++, C, Java, and python. I can learn the syntax of new coding languages very quickly.In fact I had to learn html on the spot to create this website I hope its okay. The applications I used for programming were mainly for object oriented programming so I would use Unreal engine 4 (previous version), Autocad inventor (for previous 3D printing and laser cutting projects i've worked on) , eclipse code compiler, and Autocad blueprint maker. My main goal is to become one of the best laser designers since creating really cool things has always been a passion of mine. Thank you for reading!
